# OpenKB Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a centralized Agent Kanban platform where agents use `openkb` CLI (JSON) and humans use an elegant Web UI, backed by file storage in `workspace/projects/`.

**Architecture:** Python Storage Layer shared by Typer CLI and FastAPI API; React SPA polls REST endpoints; agents never touch files directly—only CLI. Project binding via `OPENKB_PROJECT` → `.openkb-link` → `repo_path` match.

**Tech Stack:** Python 3.12+ (uv), FastAPI, Typer, pytest | React 19, Vite, TypeScript, Tailwind CSS 4, @dnd-kit, framer-motion, TanStack Query

**Spec:** `docs/superpowers/specs/2026-05-20-openkb-design.md`

**UI Quality Bar:** 优美、优雅、丝滑、舒适 — see [UI Design Direction](#ui-design-direction) before any frontend task.

## UI Design Direction

**Dual-theme product:** 同一套 UI 提供 **白天（晨光）** 与 **黑夜（静夜）** 两种主题，语义 token 一致，仅色值切换。默认跟随系统 `prefers-color-scheme`，用户可手动切换并持久化到 `localStorage`（key: `openkb-theme`）。

| 主题 | 代号 | 气质 |
|------|------|------|
| **Night** | 静夜 | 深色指挥台，长时间夜间 vibecoding |
| **Day** | 晨光 | 暖白纸感，白天明亮但不刺眼 |

**共享（两主题不变）：**
- 字体：Fraunces（标题）+ DM Sans（正文）
- Accent 暖金：`#b8956a`（day 稍深以保证 contrast）/ `#c9a87c`（night）
- 圆角：card `12px`，panel `20px`
- Motion：spring `{ stiffness: 380, damping: 32 }`
- 交互原则不变：drag lift、stagger 入场、侧栏滑出、3s 轮询

### Theme 实现约定

```css
/* tokens.css — 语义变量名，不含色值 */
:root {
  --radius-card: 12px;
  --radius-panel: 20px;
  /* colors filled by theme class */
}

/* theme-dark.css */
html[data-theme="dark"] { --bg-base: #0c0d10; /* ... */ }

/* theme-light.css */
html[data-theme="light"] { --bg-base: #f6f4f0; /* ... */ }
```

```typescript
// ThemeProvider: init from localStorage || matchMedia('prefers-color-scheme')
// ThemeToggle: sun/moon icon, cross-fade 200ms on documentElement
```

**切换动画：** 主题切换时 `html` 上 `transition: background-color 0.2s ease, color 0.2s ease`；卡片/面板跟随 CSS 变量，避免组件内硬编码色值。

**禁止：** 组件内写死 `#0c0d10` / `#fff`；禁止 day 主题用纯黑文字 `#000`（太硬）；禁止 night/day 各搞一套组件。

**Interaction principles:**
- Drag card: lift scale 1.02 + shadow bloom; column highlights on hover
- Page enter: stagger fade-up 40ms per card (framer-motion)
- STATE panel: typographic hierarchy, generous line-height 1.7
- Sidebar: slide from right 320ms ease-out, backdrop blur via `--backdrop-blur`
- Poll every 3s — TanStack Query `refetchInterval: 3000`
- Empty states: quiet line icon + one-line copy
- Theme toggle: top-right in AppShell, keyboard accessible (`aria-label`)

```python
# server/src/openkb/config.py
from __future__ import annotations
import os
from dataclasses import dataclass
from pathlib import Path

def _detect_root() -> Path:
    env = os.environ.get("OPENKB_ROOT")
    if env:
        return Path(env).resolve()
    # package at server/src/openkb → repo root is 3 levels up
    return Path(__file__).resolve().parents[3]

@dataclass(frozen=True)
class OpenKBConfig:
    root: Path
    workspace: Path
    projects_dir: Path

    @classmethod
    def load(cls) -> OpenKBConfig:
        root = _detect_root()
        workspace = root / "workspace"
        return cls(root=root, workspace=workspace, projects_dir=workspace / "projects")

    def project_dir(self, slug: str) -> Path:
        return self.projects_dir / slug
```

```python
# server/src/openkb/markdown_io.py
from __future__ import annotations
import re
from pathlib import Path
import frontmatter
from openkb.models import TaskModel, BoardColumn

SECTIONS = ["Goal", "Acceptance", "Context", "Notes"]

def _parse_sections(body: str) -> dict[str, str]:
    parts: dict[str, str] = {s.lower(): "" for s in SECTIONS}
    current = None
    buf: list[str] = []
    for line in body.splitlines():
        m = re.match(r"^## (\w+)\s*$", line.strip())
        if m and m.group(1) in SECTIONS:
            if current:
                parts[current.lower()] = "\n".join(buf).strip()
            current = m.group(1)
            buf = []
        else:
            buf.append(line)
    if current:
        parts[current.lower()] = "\n".join(buf).strip()
    acceptance_raw = parts["acceptance"]
    acceptance = [ln.strip() for ln in acceptance_raw.splitlines() if ln.strip().startswith("- [")]
    return {
        "goal": parts["goal"],
        "acceptance": acceptance,
        "context": parts["context"],
        "notes": parts["notes"],
    }

def read_task_file(path: Path, column: BoardColumn) -> TaskModel:
    post = frontmatter.load(path)
    meta = dict(post.metadata)
    sections = _parse_sections(post.content)
    filename_slug = path.stem.split("-", 1)[-1] if "-" in path.stem else path.stem
    return TaskModel(
        id=str(meta.get("id", "")),
        slug=filename_slug,
        title=str(meta.get("title", "")),
        status=column,
        priority=meta.get("priority", "P2"),
        assignee=str(meta.get("assignee", "") or ""),
        branch=str(meta.get("branch", "main")),
        created=str(meta.get("created", "")),
        updated=str(meta.get("updated", "")),
        locked_by=str(meta.get("locked_by", "") or ""),
        locked_at=str(meta.get("locked_at", "") or ""),
        lock_expires=str(meta.get("lock_expires", "") or ""),
        tags=list(meta.get("tags") or []),
        related_files=list(meta.get("related_files") or []),
        goal=sections["goal"],
        acceptance=sections["acceptance"],
        context=sections["context"],
        notes=sections["notes"],
    )

def write_task_file(path: Path, task: TaskModel, column: BoardColumn) -> None:
    meta = {
        "id": task.id,
        "title": task.title,
        "status": column,
        "priority": task.priority,
        "assignee": task.assignee,
        "branch": task.branch,
        "created": task.created,
        "updated": task.updated,
        "locked_by": task.locked_by,
        "locked_at": task.locked_at,
        "lock_expires": task.lock_expires,
        "tags": task.tags,
        "related_files": task.related_files,
    }
    acceptance_block = "\n".join(task.acceptance) if task.acceptance else "- [ ] "
    body = f"""## Goal
{task.goal}

## Acceptance
{acceptance_block}

## Context
{task.context}

## Notes
{task.notes}
"""
    path.parent.mkdir(parents=True, exist_ok=True)
    path.write_text(frontmatter.dumps(type("Post", (), {"metadata": meta, "content": body})()), encoding="utf-8")
```
